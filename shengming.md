# 本书使用说明

```text
这本书并非一本研究家庭教育的著作，也不是学生用的教学辅导书，它是一本针对家长认知水平提高的家庭教育指导纲领。

那么，这本书该如何应用？

1. 这本书尝试提供一个家长认知大框架，为家长呈现了一个全新的认知轮廓，这是一幅家庭教育知识地图，其目录可以被视作家长需要了解的家庭教育完整知识体系的大纲，里面的内容是老生家长对新生家长具体内容，能让新人少走弯路。

   过去，这种经验型的知识体系通常需要等到孩子进入大学后，家长才会逐渐洞悉架构，然而为时已晚，很多经验与领悟无法为自家孩子的成长提供及时的支持与指导，却可作为传帮带的内容提点给幼儿园、小学、中学家长们。

2. 书中都是真实学生案例，例如S爸、Z爸、L爸、W爸、H爸、Q爸和S妈、Y妈、X妈、P妈、M妈，他们就是您孩子学校的学长父母。这些身边人的成功经验具有极高的参考价值。

3. 这本书还在细节之中隐藏了大量的学习资源，只需稍微用心，家长就能在文中发现它们的蛛丝马迹。

4. 书中提供了大量的家庭教育实操方法，可供家长参考和使用。

5. 这本书还附带了一个家长群，你可以与其他家长分享请教经验，互相支持，一同前行。

6.最重要的是，这本书内容将持续更新，伴随孩子的成长，书中的经验也将不断增加，因此它将成为家长们长期受益的经验来源。

```
